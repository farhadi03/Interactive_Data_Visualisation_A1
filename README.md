# Interactive Data Visualisation: Electricity Production and CO₂ Emissions

An interactive data visualisation project that explores the relationship between CO₂ emissions and renewable electricity trends across selected countries from 1990 to 2022.

## Features

- **Line Chart**: Displays CO₂ emissions over time with the ability to toggle between total emissions and per capita emissions
- **Scatter Plot**: Shows the relationship between renewable electricity share and CO₂ per capita for a selected year
- **Interactive Controls**: 
  - Country highlighting to focus on specific countries
  - Year slider for the scatter plot
  - Metric toggle for the line chart

## Technologies Used

- **D3.js v7**: For data visualisation
- **HTML5/CSS3**: For structure and styling
- **JavaScript**: For interactivity and data processing

## Selected Countries

The visualisation includes data for:
- United States
- China
- Germany
- India
- Ireland
- Norway
- United Kingdom
- Canada
- Australia
- Brazil

## Data Sources

The project uses combined data from Our World in Data (OWID):
- CO₂ emissions data
- Energy production data
- Renewable electricity share statistics
